《DNSPOD iOS》
=================

### 说明
DNSPod 建立于2006年3月份，是一款免费智能DNS产品。 DNSPod可以为同时有电信、网通、教育网服务器的网站提供智能的解析，让电信用户访问电信的服务器，网通的用户访问网通的服务器，教育网的用户访问教育网的服务器，达到互联互通的效果。
